Git es una herramienta que nos permite controlar las versiones, podemos trabajar sin una conexioon a internet

PRINCIPALES COMANDOS DE GIT

  • GIT INIT: Este comando funciona para iniciar el proyecto en git
  • GIT ADD: Este comando permite agregar cambios al area de reparacion
  • GIT COMMIT: Este comando permite dejar uncomentario dentro de git
  • GIT STATUS: Permite visualizar el estado actual del archivo
  • GIT LOG: Permite visualizar el historial de lo realizado
  • GIT BRANCH: Permite visualizar en que rama nos encontramos
  • GIT -b BRANCH "NAME": Permite cambiar el nombre de la rama en que nos encontramos
  • GIT REMOTE ADD ORIGIN "URL": Permite entre lazar el GIT con el GitHub con el link que nos otorga
  • GIT CONFIG --GLOBAL USER.NAME: Nos permite ingresar el nombre con el que nos identificamos
  • GIT CONFIG --GLOBAL USER.EMAIL: Nos permite ingresar el correo con el que vamos estar identificados
  • GIT PUSH -U ORIGIN MAIN: Este comando funciona para activar el repositorio o el branch
  • GIT CHECKOUT: Este comando funciona para desplazarce entre las ramas creadas por git branch
